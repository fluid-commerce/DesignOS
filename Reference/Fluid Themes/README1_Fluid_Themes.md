## Getting Started

1. **Install CLI:**
   ```bash
   brew install fluid_cli
   ```

2. **Authenticate:**
   ```bash
   fluid login
   ```

3. **Pull a theme:**
   ```bash
   fluid theme pull
   ```

4. **Start development:**
   ```bash
   fluid theme dev
   ```
