01 Product Feature

Two-column hero with product image placeholder + 2×2 feature grid. Best for product launch sheets, feature overviews, and capability summaries. Default accent: blue.

Zone Layout (Top to Bottom)
  • Zone 1: Header — Fluid logo + product name tag
  • Zone 2: Hero — headline left, product image right (two-column)
  • Zone 3: Feature Grid — 2×2 cards with icon + title + description
  • Zone 4: Footer — logo + CTA button
