## Operator Design Ideas

1. **Paper tear texture** for photo edges (new asset needed)
2. **Opening quote marks** should be decorative, more opaque (25-35% not 4-10%)
3. **Uppercase NeueHaas small-caps** treatment for category labels
4. **Logo usage** — prefer brand logos at visible size over text names
5. **Carousel support** — some content better as multi-slide
6. **Research agent** for stat validation (future)
7. **Three-layer architecture** — background (decoratives), core (content), foreground (borders)
8. **Border snippets** — inject footer/header HTML verbatim, don't reconstruct each time
