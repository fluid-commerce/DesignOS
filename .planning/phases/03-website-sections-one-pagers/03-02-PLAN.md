Build the hero section as a Gold Standard compliant .liquid template. This is the proof-of-concept that validates the schema pattern works before scaling to 10+ types.

**Use `node tools/scaffold.cjs hero` to generate the starting skeleton** (per research: don't hand-roll schema). Then customize the template HTML for a hero section layout.

**Hero section elements:**
- Heading (h1): 6 settings (font_family, font_size, font_size_desktop, font_weight, heading_color, heading content)
- Subheading (h2/p): 6 settings
- Body text (p): 6 settings
- Primary CTA button: 7 settings (show, text, url, style, color, size, weight)
- Background image: image_picker
- Section settings: 5 (background_color, background_image, padding_y_mobile, padding_y_desktop, border_radius)
- Container settings: 7 (background_color, background_image, border_radius, padding_y_mobile, padding_y_desktop, padding_x_mobile, padding_x_desktop)

**Schema setting order (from Gold Standard, Weight 100):**
1. Content fields (heading, subheading, body -- each immediately followed by its 5 styling settings)
2. Button settings (show_button, button_text, button_url, button_style, button_color, button_size, button_font_weight)
3. Section settings (background_color, background_image, section_padding_y_mobile, section_padding_y_desktop, section_border_radius)
4. Container settings (container_background_color, container_background_image, container_border_radius, container_padding_y_mobile, container_padding_y_desktop, container_padding_x_mobile, container_padding_x_desktop)

**Template HTML pattern:**
```liquid
<section class="{{ section.settings.background_color }} {{ section.settings.section_padding_y_mobile }} {{ section.settings.section_padding_y_desktop }} {{ section.settings.section_border_radius }}"
  {% if section.settings.background_image != blank %}
  style="background-image: url('{{ section.settings.background_image | image_url }}'); background-size: cover; background-position: center;"
  {% endif %}>

  <div class="container {{ section.settings.container_background_color }} {{ section.settings.container_border_radius }} {{ section.settings.container_padding_y_mobile }} {{ section.settings.container_padding_y_desktop }} {{ section.settings.container_padding_x_mobile }} {{ section.settings.container_padding_x_desktop }}">

    {% if section.settings.heading != blank %}
    <h1 class="{{ section.settings.heading_font_family }} {{ section.settings.heading_font_size }} {{ section.settings.heading_font_size_desktop }} {{ section.settings.heading_font_weight }} {{ section.settings.heading_color }}">
      {{ section.settings.heading }}
    </h1>
    {% endif %}

    [... subheading, body text, button ...]

  </div>
</section>
```

**Button HTML (from Gold Standard):**
```liquid
{% if section.settings.show_button %}
<a href="{{ section.settings.button_url | default: '#' }}"
   class="btn btn-{{ section.settings.button_style | default: 'filled' }}-{{ section.settings.button_color | default: 'primary' }} {{ section.settings.button_size | default: 'btn-md' }} {{ section.settings.button_font_weight | default: 'font-medium' }}">
  {{ section.settings.button_text | default: 'Learn More' }}
</a>
{% endif %}
```
