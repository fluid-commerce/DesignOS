**Step 1: Patterns middleware in watcher.ts**
The watcher already serves `/templates/` and template-related files. Add a similar middleware block for `/patterns/`:

In the `configureServer` section, add a middleware that:
- Matches `GET /patterns/` or `GET /patterns/index.html`
- Reads `path.resolve(projectRoot, 'patterns/index.html')`, returns with `Content-Type: text/html`
- Matches `GET /patterns/*` for any other static files inside the patterns directory (CSS, JS, images)
- Serves with appropriate MIME types using the existing MIME_TYPES map

Place this middleware near the existing templates serving block. Pattern:
```typescript
// Serve patterns/ static files
server.middlewares.use(async (req, res, next) => {
  if (!req.url?.startsWith('/patterns/') && req.url !== '/patterns') return next();
  const patternsDir = path.resolve(projectRoot, 'patterns');
  const pathname = req.url === '/patterns' ? '/patterns/index.html' : req.url;
  const relative = pathname.replace('/patterns/', '');
  const fullPath = path.resolve(patternsDir, relative || 'index.html');
  // Security: prevent path traversal
  if (!fullPath.startsWith(patternsDir + path.sep) && fullPath !== patternsDir + path.sep + 'index.html') return next();
  try {
    const stat = await fs.stat(fullPath);
    if (!stat.isFile()) return next();
    const ext = path.extname(fullPath).slice(1);
    const mime = MIME_TYPES[ext] || 'application/octet-stream';
    res.setHeader('Content-Type', mime);
    createReadStream(fullPath).pipe(res);
  } catch { next(); }
});
```
