New file. Exports `scanAndSeedBrandAssets(assetsDir: string): Promise<void>`.

Logic:
- Read top-level subdirectories of `assetsDir` (each subdir = a category: brushstrokes, circles, fonts, logos, etc.)
- For each file in each category dir, run `INSERT OR IGNORE INTO brand_assets` with:
  - `id`: nanoid()
  - `name`: filename without extension
  - `category`: parent directory name
  - `file_path`: relative path like `brushstrokes/brush-texture-01.png`
  - `mime_type`: lookup from extension (.png -> image/png, .ttf -> font/ttf, .svg -> image/svg+xml, .jpg/.jpeg -> image/jpeg, .woff2 -> font/woff2, default application/octet-stream)
  - `size_bytes`: from fs.stat
  - `tags`: '[]'
  - `created_at`: Date.now()
- Use `INSERT OR IGNORE` (relies on `file_path UNIQUE` constraint) for idempotent rescans
- Import `getDb` from `../lib/db` and `nanoid` from `nanoid`
- Use `node:fs/promises` for async readdir/stat
- Skip hidden files (starting with `.`)
- Skip non-file entries (directories within category dirs)

```typescript
export interface BrandAsset {
  id: string;
  name: string;
  category: string;
  url: string;       // /fluid-assets/{file_path}
  mimeType: string;
  sizeBytes: number;
  tags: string[];
}

function rowToBrandAsset(row: Record<string, unknown>): BrandAsset {
  return {
    id: row.id as string,
    name: row.name as string,
    category: row.category as string,
    url: `/fluid-assets/${row.file_path as string}`,
    mimeType: row.mime_type as string,
    sizeBytes: row.size_bytes as number,
    tags: JSON.parse(row.tags as string),
  };
}

export function getBrandAssets(category?: string): BrandAsset[] {
  const db = getDb();
  if (category) {
    return (db.prepare('SELECT * FROM brand_assets WHERE category = ? ORDER BY name ASC').all(category) as Record<string, unknown>[]).map(rowToBrandAsset);
  }
  return (db.prepare('SELECT * FROM brand_assets ORDER BY category ASC, name ASC').all() as Record<string, unknown>[]).map(rowToBrandAsset);
}
```
