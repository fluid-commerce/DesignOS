## What Was Built

**CreationFrame** now renders in two modes based on available data:
- **src mode** (preferred): when `iterationId` is provided, iframe loads `/api/iterations/{id}/html` — eliminates 620KB+ HTML string from React state per card
- **srcDoc mode** (fallback): when only `html` string is provided — maintains backward compat with session-based canvas flows
- Both modes include `sandbox="allow-same-origin allow-scripts"` for postMessage support
- `MAX_HTML_SIZE` constant and `htmlTooLarge` guard removed entirely
- Empty state ("No preview available") shown when neither `iterationId` nor `html` is available

**VersionGrid** passes both `html={v.html}` and `iterationId={v.iterationId}` to CreationFrame — CreationFrame selects src mode automatically when iterationId is present.

**ExportActions** now fetches `?export=zip` (the ZIP bundle endpoint from Plan 01) and downloads a `.zip` file. The base64 HTML path (`?download=1`, `text/html;charset=utf-8` blob) is fully removed. Button label updated from "HTML" to "ZIP".
